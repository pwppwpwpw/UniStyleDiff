Unified diffusion-driven style transfer for images and video.

UniStyleDiff brings content and style together through a dual-branch injection design, then extends the backbone with an inter-frame consistency module to stylize full videos while preserving motion dynamics.

Dual-branch injection

Content and style are injected separately, preserving structure while driving appearance.

Style tokens

CLIP image features are projected into style tokens for text-free control.

Consistency module

ICM learns temporal coherence without retraining the full diffusion backbone.

Install

Use PyTorch, diffusers, transformers, torchvision, and Pillow.

pip install torch torchvision diffusers transformers pillow

Train Stage I

python scripts/train_stage1.py --config /path/to/stage1.json

Train Stage II

python scripts/train_stage2.py --config /path/to/stage2.json --stage1_ckpt /path/to/stage1.pt
